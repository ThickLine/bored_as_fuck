# **Flutter Starter**
Flutter starter project using Stacked. 
https://pub.dev/publishers/filledstacks.com/packages

Stack:
- stacked
- stacked_hooks
- stacked_services
- stacked_themes
- freezed
- freezed_annotation
- json_serializable
- logger
- permission_handler
- stack_trace
- shared_preferences:
- flutter_native_splash
- flutter_launcher_icons
- flutter_dotenv

------------



### Run

`flutter pub run`



### Generate
`flutter packages pub run build_runner build --delete-conflicting-outputs`
